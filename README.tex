\begin{enumerate}

\end{enumerate}

\section{Citations}

\subsection{Citation style}

We use APA-like citations because the last name of the first author appears in the text. 
This saves readers' time. (No need to hunt down number in the reference section.)
Examples:
%
\begin{itemize}
  \item You can cite parenthetically at the end of a sentence with the \verb|\citep| command  ~\citep{smith2021}.
  \item The citation can be a subject of the sentence. It read as if you say to "author and colleague...". Use the \verb|\citet| command for this: \citet{brown2020} presented some findings.
  \item Be precise with your citations when possible by, e.g., pointing to specific page number or section number. Example \verb|\citep[p. 12]{smith2021}| generates \citep[p. 12]{smith2021}.
  \item Alternatively, you can also a prefix text before parenthetical citation with a command like\\ \verb|\citep[see p. 12 of][]{smith2021}|. This results in \citep[see p. 12 of][]{smith2021}. 
  \item To cite multiple items in one parenthesis, use \citep{smith2021, brown2020}. This template does not support citing multiple items with text prefix/suffix.
\end{itemize}
%

\subsection{Reference section}
Ensure that the reference information is as complete as possible. 
Whenever available, provide Digital Object Identifier (DOI).
The bibliography style is fixed to \texttt{apalike} in the class; it does not need to be
declared separately in the \texttt{.tex} file.



\section{Word Count}

Immediately before the reference list, the class prints the word count of the body text, excluding references. The value is set in the preamble with \verb|\renewcommand{\wordcountvalue}{N}|. 
For automatic counting, run \texttt{texcount -{}-sum 0-main.tex} and copy the result.

If you use Overleaf, see \linkedfootnote{https://docs.overleaf.com/writing-and-editing/using-word-count}{how to get the word count from the GUI}. If you cannot get the word count to work in the command line, don't spend too much time figuring it out. Just compile your work in PDF, copy-and-paste it to MS Word, and use its word count feature.

\section{URLs and Hyperlinks}

When you include a URL, make it as linked text with the full address
repeated in a footnote, so the link is accessible in both digital and printed
versions.
Use the custom command \verb|\linkedfootnote{url}{text}| to automatically create this.
Example: \linkedfootnote{https://www.latex-project.org}{the \LaTeX{} project website}.

\subsection{Figures}

\autoref{fig:fullwidth} provides an example of a figure. 
When you use the command \verb|\includegraphics|, it first searches the subfolder \texttt{figures}.
If  you want to change the path, add the like \verb|\graphicspath{{your-path/}}| before \verb|\begin{document}|.
We recommend creating figure with exact physical size that it will appear on the page, such that you can just use the command \texttt{includegraphics} without any parameter in \LaTeX. In this way, you can ensure that the original figure has appropriate resolution.

\end{figure}

\section{Stylistic specifications}
If you use the template, the following settings are already applied. They are documented here for reference purposes.

\subsection{Font style}

Body text is set in \textbf{Helvetica} at \textbf{11\,pt} with a
line spacing of \textbf{1.3} lines.
Section headings are \textbf{12\,pt bold}.
Subsection headings are \textbf{11\,pt italic and underlined}. 

Do not indent the first paragraph of each section. Do indent subsequent paragraphs. Do not use ligatures.

\subsection{Margins}

The page margins are \textbf{1\,cm top}, \textbf{1.5 cm bottom} and \textbf{2\,cm left and
right}. No header is printed; the footer
area is included in the bottom margin calculation.
